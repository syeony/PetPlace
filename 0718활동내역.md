# 🐾 펫 커뮤니티 앱 - 스프린트 회고

## 💡 핵심 차별화 포인트

- **AI 실종 반려동물 매칭 시스템 (킥 기능)**
  - 품종, 색깔, 특징 자동 분석 → 매칭률 제공
  - 갤러리 업로드 금지, **실시간 촬영**으로 신뢰성 확보

## 📱 주요 기능

- 실종 동물 AI 매칭
- 품앗이 돌봄 서비스 (무료)
- 커뮤니티 기능 (자랑, 정보공유, 공구, 모임)
- AI 펫닥터
- 동네 기반 애견 카페/병원 정보 제공

## 🔍 KPT 회고

### 👍 Good

- **유진**: 아이디어를 구체화하는 과정에서 열심히 하는 모습이 좋았다
- **경호**:
  - 팀원 간의 소통 시 이해를 돕기 위한 정리와 경청이 좋았음
  - 의견도 많이 제시해줘서 좋았고, 밤에도 열심히 참여해주어 감사
- **민**: 다양한 아이디어를 제시해서 주제를 찾는 데 도움을 주어 좋았다

---

### 🔁 Keep

- **승연**: 빠지는 사람 없이 다함께 열심히 회의한 것
- **정현**: 아이디어 구상 과정에서 모두가 열정적으로 참여한 점
- **유진**: 어떤 주제든 열정적으로 참여하며 회의가 잘 진행됨
- **경호**:
  - 의견 제시와 열심히 참여하는 태도
  - 모르는 부분이 있을 때 적극적으로 알려주는 자세
- **민**:
  - 모두가 주도적으로 참여해 주제에 살을 붙인 점
  - 아이디어 구상에서 모두가 활발히 의견 제시한 점

---

### ❗ Problem

- **승연**:
  - 하나의 주제에 너무 몰입해 시간 초과되는 경향
  - → 개발 단계에서는 일정을 정해 다같이 지키는 것이 필요
- **유진**: 단호함 부족 → 주관보단 점수 제도로 의사결정하는 방식 제안
- **경호**: 주관적 의견 제시 부족 → 때론 단호한 선택이 필요
- **민**:
  - 기능 설계 시 **구현 방법**에 대해 구체적인 시간 확보 필요
- **정현**: 회의가 목적 없이 길어질 때 있음 → 목적과 예상 시간 사전 설정 필요

---

### 🚀 Try

- **유진**: 아이디어 결정 시 **점수 제도** 도입
- **승연**: 매주 **월요일마다 Jira에 일주일치 할 일 등록** (요일별 세분화 포함)
- **경호**:
  - 불만사항/건강 이슈 등 생기면 팀장·컨설턴트·코치에게 **즉시 공유**
- **정현**: 회의 전 **목적 공지 및 시간 정해두기**
- **민**: 기능 설계 시 예상 API 미리 정리
