# 펫 커뮤니티 앱 기능 명세서

## 1. 사용자 인증 및 회원 관리

### 1.1 회원가입 (필수 기능)
**담당자**: admin  
**기능 설명**: 
- 기본 정보 입력: 아이디, 비밀번호, 이메일
- OAuth 연동: 카카오, 구글
- 입력 검증: 비밀번호 제약 조건, 이메일 검증
- 동네 인증: GPS 기반
- 기존 계정 연동: 소셜 로그인 시 기존 사용자 연동
- 메타 규칙 적용

**개발 상태**: 가능

### 1.2 로그인 (필수 기능)
**담당자**: admin  
**기능 설명**:
- 아이디/비밀번호 로그인
- 자동 로그인 (토큰 인증)
- 비로그인 상태에서도 제한적 사용 가능

**개발 상태**: 가능

### 1.3 로그아웃 (필수 기능)
**담당자**: admin  
**기능 설명**: 마이페이지 내 로그아웃 버튼

**개발 상태**: 가능

### 1.4 비회원 기능
**이용 가능한 기능**:
- 홈화면 확인
- 게시판 열람 (글쓰기 불가)
- 지도 확인 (근처 애견 카페, 병원 등)
- 실종 목록 확인

## 2. 마이페이지 기능

### 2.1 프로필 (필수 기능)
**담당자**: MyPage  
**기능 설명**:
- 다정온도 표시
- 프로필 설정 및 회원정보 수정
- 임보 가능 용품 사진 등록
- 배지 시스템

**개발 상태**: 가능

### 2.2 내가 쓴 글 (기능3)
**담당자**: MyPage  
**기능 설명**:
- 작성한 게시글 모아보기
- 작성한 댓글 모아보기
- 스크랩한 글 (마킹/발도장)

**개발 상태**: 가능

## 3. 게시판 기능

### 3.1 게시판 (필수 기능)
**담당자**: Board  
**기능 설명**:
- 글쓰기, 글목록, 카테고리 분류
- **카테고리**:
  - 돌봄 (구인/구댕)
  - 나눔
  - 거래
  - 소개팅 (시간 여유 시)
  - 댕친구 (시간 여유 시)
  - 자유게시판 (댕/냥/기타)
  - 공구
- 글 찜하기 (발도장)
- 온도 시스템 (꼬순내)
- 전국 단위 서비스

**개발 상태**: 가능

### 3.2 돌봄/산책인 구인 (기능2)
**담당자**: Board  
**기능 설명**:
- 캘린더를 통한 일정 및 시간 선택
- 펫 특이사항 입력
- 비용 정보

**개발 상태**: 가능

### 3.3 임보 연계 (기능2)
**담당자**: Board  
**기능 설명**: 보호소 연계 서비스 (관리자 운영, 광고 형태)

**개발 상태**: 확인필요
