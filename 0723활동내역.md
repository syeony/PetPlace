# 펫 커뮤니티 앱 개발 프로젝트 진행상황

**날짜**: 2025년 7월 23일 (수요일)  
**프로젝트명**: 반려동물 커뮤니티 앱

---

## 📋 오늘 주요 작업 내용


### 1. ✅ API 명세서 작성 완료
총 **40여개 API 엔드포인트** 설계

#### 주요 API 카테고리:
- **사용자 인증**: 회원가입, 로그인, 중복체크, 탈퇴
- **게시판**: CRUD, 댓글, 핫글 조회
- **돌봄/산책**: 구인 게시판 전용 API
- **프로필**: 조회, 수정, 내정보 설정
- **채팅**: 1:1 채팅방, 메시지 전송/조회, 읽음처리
- **거래**: 용품 공구/나눔 게시판
- **임보연계**: 보호소 연계 서비스
- **애견호텔**: 호텔 정보 관리

#### 특이사항:
- 웹소켓 기반 실시간 채팅 구현 예정
- 관리자 전용 API 권한 관리 필요
- 비회원도 제한적 기능 이용 가능

### 2. 🔄 AI 핵심 기능 테스트 진행중
- **실종펫 매칭**: 온디바이스 AI 모델 경량화 연구
- 성능 최적화 및 정확도 개선 작업 지속

### 3. 🔄 프론트엔드 와이어프레임 설계중
#### 주요 화면 구조:
- **하단 네비게이션**: 홈, 게시판, 지도, 채팅, 마이페이지 (5탭)
- **홈화면**: 핫글, 카테고리 바로가기, 실시간 피드
- **게시판**: 카테고리별 분류, 필터링, 글쓰기
- **지도**: 애견시설 핀 표시, 리뷰 연동
- **마이페이지**: 프로필, 다정온도, 내글관리

#### 디자인 방향:
- 따뜻하고 친근한 펫 테마
- 모바일 우선 반응형 디자인
- 직관적인 UX/UI

### 4. 🔄 금요일 기획안 발표 준비 시작
#### 발표 구성안:
1. **문제정의 및 기획하게 된 배경** (3분): 기존 펫 커뮤니티의 한계점
2. **솔루션** (5분): 우리 앱의 핵심가치와 차별점

## 🎯 핵심 차별화 포인트

### 1. AI 기반 서비스
- **실종펫 매칭**: 이미지 인식으로 실종동물 찾기
- **AI 펫닥터**: 24시간 펫 상담 챗봇
- **OCR 기능**: 실종공고 자동 인식

### 2. 지역 기반 돌봄 서비스
- GPS 기반 동네 인증
- 실시간 돌봄/산책 구인
- 신뢰도 시스템 (다정온도, 꼬순내)

### 3. 사용자 친화적 접근
- 비회원도 열람 가능
- 소셜 로그인 연동
- 직관적인 5탭 구조

## 💡 개선사항 및 이슈

### API 명세서 개선 필요사항:
1. **응답 형식 표준화**: 1/0 → JSON 형태로 통일
2. **권한 관리**: JWT에 role 정보 추가
3. **페이징 처리**: 게시판 조회 API에 페이징 추가
4. **에러 핸들링**: 표준 에러 코드 정의

### 기술적 고려사항:
- 웹소켓 실시간 채팅 구현 방안
- AI 모델 온디바이스 최적화
- 이미지 업로드 및 저장 전략
- 지도 API 연동 방식 선택
